# 🔍 核心功能

`/help` - 查詢機器人顯示的全部功能介紹

`/ticket` - 開啟支援票

****AI 智能互動輔助****

**Gemini 核心驅動** : 整合 Google Gemini 大語言模型，支援自然語言對話

初步整合AI系統，以後更改專案將支援更精準的社群規則

**考官專用自動化工具**

考官可以直接透過指令變更考試成績，系統將自動同步並透過私訊 (DM) 發送美觀的 Embed 結果報告給受試者。
